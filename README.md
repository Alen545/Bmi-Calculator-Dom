# 📊 Responsive BMI Calculator (JS DOM Project)

This project was built strictly using **Vanilla JavaScript DOM manipulation**, meaning the HTML body remains completely empty, and the entire layout is rendered dynamically via script.

---

## 🚀 Features

- **100% Dynamic DOM:** Zero UI code written in the HTML file; everything is rendered dynamically through `script.js`.
- **Fully Responsive Layout:** Automatically adjusts using CSS Flexbox and media queries to look beautiful on **Mobile, Tablets, and Desktop computers**.
- **Interactive UI States:** Changes visual feedback color categories based on your results (Yellow for Underweight, Green for Normal weight, Red for Overweight).
- **Instant Input Validation:** Prevents negative numbers or empty fields from crashing the system.

---

## 🛠️ Tech Stack Used

- **HTML5** (Structural Shell)
- **CSS3** (Responsive Layouts & Media Queries)
- **JavaScript (ES6+)** (Dynamic DOM Generation & Calculation Logic)
- Developed using **Google Antigravity IDE**

---

## 📸 How It Works

1. Enter your weight in kilograms (**kg**).
2. Enter your height in centimeters (**cm**).
3. Click **Calculate BMI**.
4. The JavaScript processes the metric formula:
   $$BMI = \frac{weight\ (kg)}{height\ (m)^2}$$
5. The result box appears dynamically with your precise BMI category styling.

---

## 💻 How to Run This Locally

1. Clone or download this repository to your machine.
2. Open the project folder in your favorite editor (like **Google Antigravity** or **VS Code**).
3. Right-click `index.html` and use the **Live Server** extension to launch it in your browser.
